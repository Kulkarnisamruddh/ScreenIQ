# ScreenIQ 🎓

AI-powered bulk resume screener built for 
college placement officers and HR teams.

## Live Demo
https://campus-screen.vercel.app

## What it does
- Upload 50+ resumes at once
- Paste a job description
- Get AI-ranked candidates with scores, 
  strengths, weaknesses and red flags
- Export results as CSV

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Python + FastAPI
- AI: Groq API (LLaMA 3.3 70B)
- PDF Parsing: pdfplumber
- Deployed: Vercel + Railway
